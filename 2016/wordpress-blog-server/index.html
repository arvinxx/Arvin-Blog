#安装 Nginx
安装完 nginx 以后，正常的操作是创建虚拟主机，虚拟主机允许我们在同一台服务器上运行多个网站。正常情况下，我将我的域名解析到我的服务器之后，一般是只能访问 Nginx 初始指定的目录的。
而有了虚拟主机，可以将多个域名同时解析到我们这台服务器上，然后通过 Nginx 为不同的域名绑定不同的目录。访问这个域名的时候，会访问对应目录里面的文件。
